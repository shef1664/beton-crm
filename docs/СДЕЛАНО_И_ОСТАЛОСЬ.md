## 🔧 ЧТО ВАМ НУЖНО СДЕЛАТЬ (5 минут):

### ШАГ 1: Получите НОВЫЙ токен

1. Откройте **@BotFather** в Telegram
2. Напишите: `/mybots`
3. Выберите: **@pulsar_daily_report_bot**
4. Нажмите: **Bot Settings**
5. Нажмите: **Token**
6. Нажмите: **Revoke current token**
7. BotFather создаст **НОВЫЙ токен**
8. **Скопируйте его** (выглядит как: `123456789:ABCdef...`)

### ШАГ 2: Обновите Mini App URL

**В ТОМ ЖЕ BotFather** (после получения токена):

1. Всё ещё в настройках **@pulsar_daily_report_bot**
2. Нажмите: **Bot Settings**
3. Посмотрите есть ли пункт **Menu Button**
4. **ЕСЛИ ЕСТЬ:**
   - Нажмите: **Menu Button** → **Configure Menu Button**
   - Отправьте: `https://shef1664.github.io/smart-fleet-app`
   
5. **ЕСЛИ НЕТ Menu Button:**
   - Нажмите: **Description** → введите описание (любое)
   - Напишите боту `/start` в Telegram
   - Должна появиться кнопка Mini App

### ШАГ 3: Обновите Railway (если нужно)

1. Откройте: https://railway.app
2. Войдите в аккаунт
3. Найдите проект PULSAR
4. Откройте **Variables**
5. Обновите `BOT_TOKEN` на новый из Шага 1
6. Откройте **Deployments** → **Redeploy**
