1. В Telegram найдите **@BotFather**
2. Или перейдите: https://t.me/BotFather

### Шаг 2: Выберите вашего бота
1. Напишите: `/mybots`
2. Выберите: **@smart_fleet_kembot**

### Шаг 3: Обновите Menu Button URL
1. Нажмите: **Bot Settings**
2. Нажмите: **Menu Button**  
3. Нажмите: **Configure Menu Button**
4. BotFather попросит: "Send me the URL..."
5. **ОТПРАВЬТЕ ЭТОТ URL:**
   ```
   https://shef1664.github.io/smart-fleet-app
   ```
6. BotFather спросит текст кнопки - оставьте как есть или напишите: `Открыть приложение`

### Шаг 4: Очистите кэш Telegram
**ОБЯЗАТЕЛЬНО сделайте это после обновления URL:**

**Способ 1 (простой):**
1. Полностью закройте Telegram
2. Откройте Telegram
3. Найдите **@smart_fleet_kembot**
4. Нажмите кнопку "Меню" слева от поля ввода

**Способ 2 (если не помогло):**
1. Удалите чат с @smart_fleet_kembot полностью
2. Найдите бота через поиск
3. Напишите `/start`
4. Нажмите кнопку Mini App
