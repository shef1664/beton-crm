# Project Scope

This repository currently contains two different code paths.

Active concrete sales funnel:
- `backend/`
- `landing-main/`
- `variants/`
- `baserow/`

Legacy PULSAR archive:
- `bot/`
- `miniapp/`
- parts of `docs/` that mention PULSAR, fleet roles, or daily reports

If you work on lead capture, amoCRM, pricing, calculator logic, or landing
experiments, stay inside the active concrete sales funnel paths.

The PULSAR files are kept only so the historical code is not lost.
